## Credit Card Sales Analysis Dashboard (Power BI)

## Project Overview
This project is a Power BI dashboard built to analyze credit card customer financial data and generate insights on spending behavior, credit utilization, payment trends, and risk indicators.

## Tools Used
- Power BI Desktop
- Power Query
- DAX
- Excel/CSV Dataset

## Key Insights
- Category-wise spending distribution
- Credit utilization trends
- Payment behavior (On-time vs Late)
- Outstanding balance and risk monitoring
- Customer segmentation using demographic data

## Outcome
This dashboard helps in identifying high-risk customers and supports data-driven financial decision-making.
